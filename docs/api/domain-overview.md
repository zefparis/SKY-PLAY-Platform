# Domain Overview

## Auth
signup, login, refresh token, reset password, email verification

## Users
profile, country, avatar, preferences, player stats

## Gaming Accounts
Activision ID, PSN, Xbox, Steam, Epic

## Challenges
create, join, invite, start, submit result, confirm result, dispute

## Tournaments
brackets, rounds, seeding, standings

## Wallet
balance, ledger, hold funds, release funds, payout, withdrawal

## Chat
global, challenge room, direct messages

## Notifications
in-app, email, push

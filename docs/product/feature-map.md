# Feature Map

## Public Platform
- Landing page
- Challenge catalogue
- Tournaments list
- Leaderboard
- Login / signup / forgot password
- Player profile / stats / achievements
- Wallet / deposit / withdraw / history
- Challenge chat / DM / notifications
- Support / FAQ

## Admin
- User management
- Gaming account review
- Match review / disputes
- Payments / payouts / KYC hooks
- Tournament management
- Moderation / bans / reports
- Analytics dashboard

## Core Business Rules
- Les joueurs utilisent des jeux externes, la plateforme orchestre les challenges.
- Les challenges sont payants via wallet.
- Les gains reviennent dans le wallet.
- Les litiges passent par un workflow dispute -> review -> resolution.
- PWA first pour mobile au Cameroun.
